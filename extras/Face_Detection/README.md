
# Training data for Facial Recognitions

Simple trainer for training faces. This capture the images data into a folder with a specific names for model construction.

# Training Process

Start the code with following:

    python face_detection_training.py -n <Name>


To capture the images, press the `a` button on keyboard. This will put the images in the following directory `training_data/<Name>/`.

To adjust the confidence interval:

    python face_detection_training.py -n <Name> -c <confience>


To quit the software, press the `q` button.
